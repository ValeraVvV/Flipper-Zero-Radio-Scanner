Модификация приложени Радио Сканер (Radio-Scanner) для Flipper Zero
## Основано на [Flipper-Zero-Radio-Scanner](https://github.com/RocketGod-git/Flipper-Zero-Radio-Scanner) от RocketGod-git

Дополнения и изменения:

1. **Стартовая частота**
   при запуске запрашивается стартовая частота 

![](screenshots/Screenshot_s.png)

ее можно пропустить и продолжить дальше.

2. **Антенна**
   запрашивается какая будет использоваться антенна

![](screenshots/Screenshot-antena.png)

если указана внешняя антена, но ее нет по факту, в этой версии происходит креш.
возможно, позже исправлю этот недочет.

3.**Технические изменения:**  
Заменен пресет модуляции с `2FSK` на `4FSK`
- увеличивает скорость приема данных
- улучшает помехоустойчивость
- чище звук
